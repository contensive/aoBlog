In each of these folders, place documents in folders exactly as you want them copied to the server.


cdnFiles
publically available files. Where customer content is stored

helpFiles
ignore this folder

layoutFiles
When creating layouts for addons, put html files in this folder
and any assets like images, css, etc in subfolders you expect 
to be stored in the wwwroot. This works just like wwwFiles except
the html files are not stored in the wwwRoot but instead they
are stored in layout recrods.

wwwFiles
Add files you want added to the www root
